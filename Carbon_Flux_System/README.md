#  碳绘喀斯特：县域固碳评估与情景模拟沙盘

本项目是一款立足于喀斯特石漠化治理业务需求的政务决策与生态规划辅助工具。通过集成多元非线性特征矩阵与边际贡献度量化算法，系统实现了固碳潜力的精准核算与 CCER 碳资产的经济评估。

为降低基层业务人员的使用门槛，系统最新引入了**基于大语言模型（LLM）的智能生态助理**，打通了从“冰冷数据核算”到“自然语言决策指导”的最后一公里。

---

##  系统架构

项目采用三层模块化设计，具备良好的扩展性与可维护性：

* **Config 层**: `config.py` 存储全局转换系数、植被映射字典、基准碳价及页面基础配置。
* **Utils 层**: 
  * `model_handler.py`: 核心生态测算引擎的异步加载与特征矩阵推演。
  * `visualizer.py`: 基于 Matplotlib 与 Altair 的动态推演图表（如 SHAP 瀑布图、雷达图等）。
  * `economics.py`: 环境经济学核算公式库，实现“固碳量(C) -> 二氧化碳当量(CO₂e) -> 经济价值(CNY)”的链路转换。
  * `logger.py`: 系统运行日志记录模块。
* **App 层**: `app.py` 响应式前端交互主界面。负责页面路由（多 Tab 切换）、全局状态管理（Session State）以及外部大模型 API 的流式调度。

---

## 核心功能模块

本沙盘系统提供三大核心业务视图：

### 1. 单点精细诊断 (Micro-Level Assessment)
* **动态调节**：支持通过交互式滑块调节气象条件（温、湿、水、辐射）与地貌特征（坡度、土层厚度、裸岩率）。
* **实时核算**：毫秒级输出“年度固碳潜力”与“生态资产综合损益”。
* **反向推演**：锁定环境参数，自动执行全路径矩阵扫描，输出适生植被（乔/灌/草）的最优规划推荐及风险预警（如：浅土层种植乔木预警）。

### 2. 区域批量测算 (Macro-Level Batch Calculation)
* **大批量导入**：支持上传 `.csv` 或 `.xlsx` 格式的区域林班调查数据，系统自动执行表头匹配与缺失值合规性校验。
* **全景报表**：一键执行现状资产核算与最优规划推演，支持核算结果与规划建议书的报表导出。

### 3. 智能生态助理 (AI Ecological Assistant) 
* **专业决策智囊**：内嵌专属生态专家 AI 助理，基于自然语言解答碳汇核算逻辑与喀斯特地貌修复建议。
* **隐性业务红线**：底层强注入喀斯特生态学诊断逻辑，AI 在回答时会自主遵循“土层厚度与植被生存红线”等业务法则，杜绝“机器幻觉”提供无效建议。

##  核心技术亮点

* **极速流式交互 (Streaming UI)**：大模型对话模块采用异步流式输出与加载动画（Spinner），彻底消除算力排队时的视觉阻塞感，实现 ChatGPT 般的打字机级顺滑体验。
* **滑动窗口记忆算法 (Sliding Window Memory)**：为解决长对话带来的“上下文溢出”与“Token 成本膨胀”，系统自主实现了会话清理策略。动态拦截并截取最新 10 轮对话，同时物理隔离并永久保护核心系统人设（System Prompt）不被遗忘。
* **防御级状态管理 (Defensive State Management)**：针对 Streamlit 多页面架构易引发的上下文丢失异常（`AttributeError`），采用全局字典级状态锁定与容错兜底机制，保障用户在不同业务模块间自由穿梭时，系统状态与 AI 记忆永不断线。
