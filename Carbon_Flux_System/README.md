# 碳绘喀斯特：县域固碳评估与情景模拟沙盘

本项目是一款立足于喀斯特石漠化治理业务需求的政务决策工具。通过集成多元非线性特征矩阵与边际贡献度量化算法，实现了固碳潜力的精准核算与 CCER 碳资产的经济评估。

## 系统架构

项目采用三层模块化设计，具备良好的扩展性与可维护性：

* **Config 层**: `config.py` 存储全局转换系数、植被映射及页面配置。
* **Utils 层**: 
  * `model_handler.py`: 核心测算引擎异步加载与矩阵推演。
  * `visualizer.py`: 基于 Matplotlib 与 Altair 的动态推演图表。
  * `economics.py`: 环境经济学核算公式库（C -> CO2e -> CNY）。
  * `logger.py`: 系统运行日志记录模块。
* **App 层**: `app.py` 响应式前端交互界面。
